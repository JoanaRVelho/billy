Billy


Billy is an open source Java library developed by Premium Minds. Its aim is to provide applications for creating and managing invoices, and credit notes, as well allowing their export to different file types. Billy also allows the creation and export of SAF-T PT(Standard Audit File for Tax purposes portuguese version) files.

Billy is currently at version 1.0, and is now available to the open source community.


Billy is currently still being developed, so it may still contain errors!

Getting Started

A complete getting started guide is available, but for those looking to contribute for the project, make sure you have installed Java 6, Git, and Maven 2.

Maven Dependency

After you have all the necessary software, you need to add Bill's Maven dependency to the pom.xml file of your project. To do that you just add: 

<dependency>
        <groupId>com.premiumminds</groupId>
        <artifactId>billy</artifactId>
        <version>${billy.version}</version>
</dependency>
