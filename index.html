Billy


Billy is an open source Java library developed by Premium Minds. Its aim is to provide applications for creating and managing invoices, and credit notes , as well as allowing their export to different file types. Billy also allows the creation and export of SAF-T PT(Sandart Audit File for Tax purposes portuguese version) files.

Billy is currently at version 1.0, and is now beginning to the open source community.


Billy is currently still being developed, so it may still contain errors!

Getting Started

A complete getting started guide is available, but for those looking to start running quickly, make sure you have installed Java 7, at least Postgres 9.1, Git, and Maven 2.

Download

After you have all the necessary software, you need to download the Billy aplication, to do that you have two options:
To get the project you must have to be registered at Git

  1. Download the latest stable release,as a zip file from Billy's GitHub
  2. Clone the git repository: 
$ git clone https://github.com/premium-minds/billy.git
and checkout the release that you want.

Install

To install Billy, you go the folder that you have just downloaded, build the project, in each folder inside billy's folder execute:

$ mvn install
After you do this you may start to develop your application, using Billy.
